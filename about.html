About Me

안녕하세요, 개발자 재성입니다 👋

대기업 IT 자회사에서 R&D 업무를 담당하고 있으며, 엔터프라이즈 비즈니스 시스템 개발을 전문으로 합니다. ERP-그룹웨어 연동, 배치 API 미들웨어, RPA 기반 자동화 등 다양한 프로젝트 경험을 보유하고 있습니다.

주요 기술 스택

Backend

  • Spring Boot, MyBatis, JPA
  • Java 17+, Kotlin
  • MSSQL, PostgreSQL, Oracle
  • RESTful API 설계

Frontend & Mobile

  • Flutter, Dart
  • Riverpod 상태관리
  • HTML, CSS, JavaScript
  • Webix UI 프레임워크

DevOps & Tools

  • Prometheus, Grafana 모니터링
  • Git, GitHub Actions
  • Docker, Tomcat
  • AI 도구 (Claude, Cursor)

현재 관심사

AI 활용 개발 생산성 향상에 큰 관심을 가지고 있습니다. 특히 Claude Code를 활용한 개발 워크플로우 최적화와, 중견기업 대상 AI 기반 개발 표준 컨설팅에 대한 연구를 진행 중입니다.

또한 Flutter + Supabase 기반의 캐주얼 게임 대량 생산 시스템(BitRoom Factory)을 개인 프로젝트로 진행하고 있으며, 이를 통해 모바일 앱 개발과 수익화에 대한 실험을 하고 있습니다.
